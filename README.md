# Todo Application
Todo application developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add todo
- Edit Todo
- List Todo
- Delete Todo
- Login
- Signup
- Responsive layout

## Technology stack
- Java 8+
- JSP and Servlet
- JDBC
- MySQL database
- Intellij IDE
- Tomcat server 8+
